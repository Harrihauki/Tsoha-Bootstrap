## Työn aihe

Jalkapallon elo-rankkausjärjestelmä. Järjestelmä laskee ja pitää yllä shakista tutun elo-järjestelmän mukaelman avulla mallia jalkapallojoukkueiden keskinäisistä voimasuhteista.
